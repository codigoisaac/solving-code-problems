# Question

Write a function that receives two string parameters `s1` and `s2` and returns true if they are a permutation of the other, otherwise it should return false.

---

# Sample

Returns true (permutations):

```
ab, ba
abc, acb, bac, bca, cab, cba
aab, aba, baa
```

Returns false:

```
ab, aa
ab, aba
aab, abb
```
